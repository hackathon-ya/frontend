### Запуск в режиме разработки
1. Склонируйте репозиторий
   ```bash
   git@github.com:hackathon-ya/frontend.git
   ```
2. Перейдите в директорию с проектом:
   ```bash
   cd frontend/
   ```
4. Установите зависимоти
   ```bash
     npm i
   ```
5. Запустите
   ```bash
     npm run dev  
   ```
